# Cart Service (Node.js)

This is the Node.js implementation of the Cart Service.

## Running the service

### Prerequisites
- Docker
- Node.js 18

### With Docker
```bash
docker build -t polyglot/cart-service-nodejs .
docker run -p 8080:8080 polyglot/cart-service-nodejs
```

### Locally
```bash
npm install
npm start
```
